CSS Padding Reference Guide


CSS Padding Syntax and Effects

The CSS padding properties are used to generate space around an element's content, inside of any defined borders.

With CSS, you have full control over the padding. There are properties for setting the padding of each side of an element (Top, right, bottom, and left)

Property Name Effect
padding The padding property is a shorthand property for the following individual padding properties. When using only one parameter, the padding amount used cascades to all sides of the element

So here's how it works...

If the padding property has four values:

  • padding: 25px 50px 75px 100px;
    • top padding is 25px
    • right padding is 50px
    • bottom padding is 75 px
    • left padding is 100px

Notice that the values are assinged from the top and proceeding clockwise, ending at the left side.

You can address these properties directly as well:

Property name Effect
padding-top Controls the amount of padding on the top of the element
padding-right Controls the amount of padding on the right side of the element
padding-bottom Controls the amount of padding on the bottom of the element
padding-left Controls the amount of padding on the left side of the element

And if the padding property has three values:

  • padding: 25px 50px 75px;
    • top padding is 25px
    • right and left paddings are 50px
    • bottom padding is 75px

Notice that when padding uses 3 values, the sides share a value.

Lastly, if the padding property has only 2 values:

  • padding: 25px 50px;
    • top and bottom paddings are 25px
    • right and left paddings are 50px

Notice that when padding uses 2 values, both the sides and the horizontal boundaries share values respectively.
